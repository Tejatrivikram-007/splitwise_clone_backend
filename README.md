# FastAPI Splitwise Clone

This repository contains a FastAPI-based backend application that replicates core functionalities similar to Splitwise — managing shared expenses and balances. It uses PostgreSQL for data storage and is containerized using Docker.

## Features

- RESTful APIs built with **FastAPI**
- Database integration using **PostgreSQL**
- ORM handled via **SQLAlchemy**
- Data validation with **Pydantic**
- Asynchronous server with **Uvicorn**
- Containerized using **Docker** and **Docker Compose**

## Getting Started

### Prerequisites

- Docker
- Docker Compose
